# SyneSpark Workspace Guide

- Workspace: Romulosignatures
- Agent mode defaults live in `.synespark/settings.local.json`.
- Persistent synapse memory graph lives in `.synespark/synapse.local.json`.
- Synapse orchestration sidecar lives in `.synespark/synapse.orchestrator.local.json`.

## Recommended Workflow

1. Keep PLAN mode for architecture and decomposition.
2. Switch to ACTION mode for implementation.
3. Keep Proactive Mode enabled when you want strict pre-delivery verification.
4. Use branch chat to fork context and run parallel task threads.
5. Review `/improve` and `/experiments` periodically.
6. Run `/hardening` before production changes.

## Notes

- This folder is local project metadata for SyneSpark.
- You can commit it to share team defaults, or ignore it for personal preferences.
